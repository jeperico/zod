
# ZOD

## Init
  
Start a Node Project:

``` bash

npm  init  -y

```

***

Install TypeScript:

``` bash

npm  install  --dev  typescript

```

***
Start tsc:

``` bash

npx  tsc  --init

```

***
Install zod:

``` bash

npm  install  zod

```

## Run

Run `PersonSchema.ts`:

``` bash

ts-node PersonSchema.ts 

```
